THE APP NAME SEEMS TO BE @lucaspennice/SkillTrees

Command to profile in hermes: npx react-native profile-hermes .

BE SURE TO INCREASE THE APP AND ANDROID RELEASE VERSION

TO CREATE AN apk IN ADROID
eas build --profile preview  --platform android

To create a development build for android phone/emulator
eas build --profile development --platform android

CREATE A NEW ANDROID BUILD

eas build --platform android

UPLOAD THE NEW UPDATE

eas submit -p android --latest

☢️ BEFORE RELEASING A NEW VERSION ☢️

- Using the list of features added/modified in this version, test all of them to check if they are still working correctly
